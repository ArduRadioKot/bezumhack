# Bezumhack CTF 2026

## Категории задач

### Pwn

- Бинарные уязвимости
- Переполнение буфера
- ROP-цепочки
- Форматные строки

### Crypto

- **Weak Hash** — взлом MD5 хеша пароля админа
- **Predictable Session ID** — предсказание сессионных ID
- Классические шифры
- RSA уязвимости

### Forensics

- **Database Leak** — восстановление удалённых записей из SQLite
- **Log Analysis** — анализ логов сервера
- Анализ сетевого трафика
- Memory forensics

### Osint

- Разведка по открытым источникам
- Геолокация по фото
- Поиск информации о доменах

### Reverse

- Реверс-инжиниринг бинарных файлов
- Анализ malware
- Крэкме

### Network

- Сетевые уязвимости
- Анализ пакетов
- ARP/DNS spoofing

### Admin

- Системное администрирование
- Конфигурация сервисов
- Hardening систем

### Misc/Joy

- **Hidden Endpoint** — поиск скрытых debug endpoints
- **HTTP Headers** — анализ кастомных заголовков
- QR-коды
- Нестандартные задачи

### Web

- **SQL Injection** — обход фильтров цен в поиске товаров
- **XSS** — внедрение скрипта в отзывы
- **IDOR** — доступ к чужим заказам
- **Race Condition** — двойная трата средств
- **Broken Access Control** — перехват корзины
- **SSRF** — доступ к внутренним сервисам
- **Command Injection** — RCE через обработку изображений
- **Path Traversal** — чтение файлов за пределами директории
- **JWT Weak Secret** — подделка токена админа
- **Insecure Deserialization** — RCE через pickle

### Stegano

- Скрытые данные в изображениях
- Аудио-стеганография
- LSB методы

### PPC

- Алгоритмические задачи
- Оптимизация кода
- Автоматизация решений
